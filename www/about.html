About this project

A quick tour of what's actually running behind this page.

What it is

webserv is an HTTP/1.1 server written from scratch in C++98 for 42 School. A single epoll instance drives every socket and CGI pipe, so the whole server runs non-blocking on one thread — no worker pool, no blocking reads.

What it supports

  • Static file serving with MIME-type detection
  • Directory listing (autoindex)
  • File uploads via POST, multipart or raw
  • File removal via DELETE
  • CGI execution (Python, PHP) with proper SCRIPT_NAME/PATH_INFO splitting
  • Chunked transfer-encoding on input
  • Keep-alive connections
  • Cookie-based sessions
  • Configurable error pages, body-size limits, and redirects
  • Timeouts for idle clients and runaway CGI processes

This page itself

You're looking at a static file served from www/about.html through the default location / route — nothing fancier than open() + read() and a Content-Type: text/html header.
