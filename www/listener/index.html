Secondary virtual host

This server block listens on 0.0.0.0:8081 with its own root, ./www/listener — a separate document root from the main host on port 8080.

Uploads

location /uploads — GET POST DELETE, autoindex on

/uploads/ — directory listing for this host

Redirect

location /redirect — GET, return 301 /

/redirect — permanent redirect back to this page.

Error pages

Only 404 and 500 have custom pages configured for this host (same files as the main host, since error_page paths are plain filesystem paths).

  • /this-page-does-not-exist — 404
